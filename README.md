# CodeAlpha - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project is part of the CodeAlpha Data Analytics Internship.

The objective is to perform Exploratory Data Analysis (EDA) on the Titanic dataset to understand the data, identify patterns, detect missing values, and visualize important insights.

## 📂 Dataset

Dataset: Titanic Dataset

## 🛠 Tools & Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Analysis Performed

- Dataset Information
- Missing Value Analysis
- Statistical Summary
- Survival Count Visualization
- Gender Distribution
- Age Distribution
- Correlation Heatmap

## 📈 Results

The project successfully identified:
- Passenger survival distribution
- Gender-wise passenger count
- Age distribution
- Correlation between numerical features
